# Pacagroup Ecommerce

En este proyecto se aplica la arquitectura DDD *(Infraestructura, Dominio y Aplicacón)* y posteriormente se añade la capa **Transversal** y de **Servicio** para pasar hacer el modelo de arquitectura N-Capas. 

Se práctica también los principios SOLID e inyección de dependencias utilizando .NET Core 3.
